# ROTRA Mascot — Rion (Clean 3D Style)

- Clean stylized 3D character
- White background

- Premium SaaS mascot style (Apple / Notion / Stripe)

Name: Rion
Role: System operator / queue controller
Personality: Calm, confident, efficient

- Hair: modern, slightly stylized sweep

## Body Proportions
- Slightly stylized (not realistic, not chibi)
