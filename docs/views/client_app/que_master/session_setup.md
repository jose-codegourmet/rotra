## Automatic Queueing settings

When configuring a Que Session, hosts may enable **Automatic Queueing** and set the default matchmaking mode, operating level, and related thresholds.

Full setting definitions, defaults, and allowed values: [`../../../business_logic/client_app/automatic_queueing.md`](../../../business_logic/client_app/automatic_queueing.md) §22.

Recommended UI placement: collapsible **Automatic Queueing** section in Session Setup (Draft/Open) and editable via **Settings** in the Que Master Console while Active.
